# Robô de Combate a Incêndio com Arduino

Uma versão DIY de robô que detecta chamas e automaticamente se dirige até elas para extinção.

## Visão Geral

Este projeto demonstra como montar um robô autônomo que utiliza sensores de chama, microcontrolador Arduino, motores de tração, bomba de água e servo-motor para localizar e apagar focos de fogo pequenos. Quando o sensor de chama detecta fogo, o robô se movimenta em direção a ele, aciona a bomba de água e o bico controlado pelo servo pulveriza água para extinguir a chama.

### Funcionalidades

- Detecção de chama através de sensor de calor.

- Navegação automática em direção ao fogo (com base na leitura de múltiplos sensores de chama ou posição da chama).

- Acionamento de sistema de extinção: bomba de água + bico controlado por servo.

- Parada ou modo standby após a extinção.

- Possibilidade de ampliação: maior tanque de água, sensores adicionais, maior autonomia.
